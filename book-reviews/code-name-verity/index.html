"Code Name Verity"

by Elizabeth Wein · Code Name Verity #3

  • Read on 2026-04-14
  • Rating: ️️️️️
  • Format: 🎧 (10 hours 3 minutes)

This is a fun, quick read. The imagery of the stories told by these two best friends in WWII is top notch. One a pilot, the other a spy. When I finished the book, I immediately wanted to restart it, since I knew I had missed things, and would appreciate what had happened even more the second time through. This was a book that a book club I'm in read before I joined the club - and they have continued to reference it across the years. It felt like obligatory reading to understand references more. I'm glad I'll be more clued in - but it's worth reading on its own.
